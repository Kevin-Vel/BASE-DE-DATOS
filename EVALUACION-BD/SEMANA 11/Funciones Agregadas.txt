Las funciones agregadas son consultas para una mayor eficiencia en el sistema y poder shorrar recursos

Comando SUM: Te da el total de una columna de tu tabla
SELECT SUM(columna) AS Total
FROM tabla;

Comando Count: Cuenta el numero de registros en la tabla
SELECT COUNT(*) AS Cantidad
FROM tabla;
y Para valores nulos hacemos lo siguiente:
SELECT COUNT(columna) AS CantidadNoNula
FROM tabla;

Comando AVG: Calcular el promedio de una colummna de tu tabla
SELECT AVG(columna) AS Promedio
FROM tabla;

Comando Max: obtiene el valor maximo de una de tus columnas
SELECT MAX(columna) AS Maximo
FROM tabla;

Comando Min: Obtiene el valor minimo de una de tus colummnas
SELECT MIN(columna) AS Minimo
FROM tabla;

Group By: Se usa para agrupar registros con valor de columna en comun se utiliza junto las funcines agregadas
SELECT columna_agrupada, SUM(columna_valor) AS Total
FROM tabla
GROUP BY columna_agrupada;

Having: Se usa para filtrar los resultados este se puede usar junto al Group by especificamente para funciones agregadas
ELECT columna_agrupada, SUM(columna_valor) AS Total
FROM tabla
GROUP BY columna_agrupada
HAVING SUM(columna_valor) > 1000;
